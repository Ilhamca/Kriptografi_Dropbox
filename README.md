# Kriptografi — Streamlit Demo

A small Streamlit app demonstrating a Caesar cipher.

Run locally:

```powershell
pip install -r requirements.txt
streamlit run main.py
```

Files:
- `main.py` — Streamlit app entrypoint
- `src/crypto.py` — Caesar cipher utilities
- `tests/test_crypto.py` — unit tests
